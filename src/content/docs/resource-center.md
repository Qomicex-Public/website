资源中心是一个一站式资源浏览器，让你无需离开启动器即可搜索和安装各种 Minecraft 资源。

#### 支持的平台

- **Modrinth** — 搜索和浏览 Modrinth 平台上的模组、资源包、光影包和数据包。
- **CurseForge** — 访问 CurseForge 的海量资源库。
- **Mcmod 百科** — 中文模组百科，提供详细的模组介绍和下载链接。
- **FTB** — 浏览和安装 FTB 整合包。

#### 使用方法

1. 点击侧边栏的「资源中心」进入。
2. 在搜索框输入关键词，选择平台和资源类型。
3. 浏览结果列表，点击资源查看详情。
4. 在详情页点击「安装到实例」，选择目标实例即可一键安装。
