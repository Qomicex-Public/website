从[下载页面](/download)获取对应系统的安装包。

![下载页面](/tutorial/download.jpg)

- **Windows** — 运行 `.msi` 安装包，按向导完成安装。完成后从开始菜单启动。
- **macOS** — 挂载 `.dmg` 文件，将 Qomicex 拖入 Applications 文件夹。首次启动需在「安全性与隐私」中允许打开。
- **Linux** — 运行 `.AppImage`（`chmod +x` 后直接运行），或通过包管理器安装 `.deb` / `.rpm` 包。

启动器首次运行时会自动检测 Java 环境并引导初始设置。
