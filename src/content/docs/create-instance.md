游戏实例是 Qomicex 的核心概念，每个实例代表一个独立的 Minecraft 安装。

1. 点击侧边栏的「实例」进入实例列表页面。
2. 点击「创建实例」按钮。
3. 选择 Minecraft 版本（支持 1.6 至最新版本，包括快照）。
4. 选择模组加载器：Forge、Fabric、Quilt、NeoForge 或 OptiFine。
5. 选择加载器版本（启动器会自动推荐兼容版本）。
6. 点击「安装」，启动器将自动下载所需文件并创建实例。

![创建实例](/tutorial/create-instance.jpg)

安装完成后，实例卡片上会显示版本信息和已安装的加载器图标。点击实例即可进入详情页面。
