Qomicex 需要 Java 17 或更高版本。首次启动时启动器会自动检测系统中的 Java。你也可以在设置中手动配置：

1. 点击侧边栏的「设置」进入设置页面。
2. 在「Java 运行时」部分，点击「浏览」手动选择 Java 可执行文件路径。
3. 或点击「下载 Java」让启动器自动下载并配置一个兼容的 Java 版本。
4. 你还可以为每个游戏实例单独指定 Java 参数（如内存分配、JVM 参数等）。

<div class="rounded-lg border border-yellow-500/20 bg-yellow-500/5 px-4 py-3">
**提示：**建议为大型 Mod 包分配 4GB 以上内存。你可以在实例设置中调整 `-Xmx` 参数。
</div>
