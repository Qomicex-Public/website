Qomicex 集成了基于 EasyTier 的内网穿透联机功能，让你无需公网 IP 或端口转发即可与好友畅玩。

#### 创建联机房间

1. 在「联机」页面点击「创建房间」。
2. 选择需要联机的游戏实例。
3. 启动器会自动建立虚拟网络连接。
4. 将房间码分享给好友即可。

#### 加入联机房间

1. 在「联机」页面点击「加入房间」。
2. 输入好友分享的房间码。
3. 连接成功后，好友的虚拟 IP 会显示在列表中。
4. 在 Minecraft 多人游戏中添加该 IP 即可加入。

**注意：**所有参与者都需要安装 Qomicex 启动器并进入联机页面。联机性能取决于网络状况。
</div>
